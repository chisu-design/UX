CatFinder

CatFinder is a web app that allows people to share their location on cats they come across and upload cat pictures.

Please turn on location service when using the app.


Project Statement

As a cat lover, I always want to know if a cat is nearby whenever I go. This led to the creation of this web application so I can pinpoint where I last saw the cat and upload a picture on how it looks like. Originally, I’ve created this app for public use, but I figure people might start to chase stray cats. This might not be a good thing for cats, so now I’ve created this app for personal use. This app also demonstrates my front-end development skills.


Technologies

• Front-End Development
• Firebase: Cloud function and storage
• JavaScript
• HTML5
• CSS3
• Bootstrap 4
• Geolocation API


Wireframe

Code

I used Firebase, a Backend-as-a-Service that allows me to store data on the cloud. The database contains images, information on the cat name, and GPS coordinates. I used HTML5, CSS3, and jQuery to make the layout a functionality. It has a responsive layout based on what device the user is using.
